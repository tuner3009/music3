# Muzora AI (Vite + React + Supabase)

Premium dark AI music studio MVP scaffold.

## Stack
- React + Vite + TypeScript + Tailwind
- React Router + Framer Motion + Lucide + Recharts
- Supabase JS + Supabase Edge Functions
- Stripe Checkout + Billing Portal

## Run
```bash
npm install
npm run dev
```

## Deploy
- Static frontend: Netlify or Vercel
- Supabase Edge Functions handle all secrets

## Security
- Never expose Suno/Stripe/Supabase service role keys in the frontend.
